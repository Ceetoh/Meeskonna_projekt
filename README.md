# Snake game

## Highlights
It was supposed to be group project, but i did it alone

## How to run

Download Python.
Install Pygame. To install it, type the following text in terminal: pip3 install pygame

## How to play

You are a snake and you need to collect food.

Controls

⬇️ Move down

⬆️ Move up

➡️ Move right

⬅️ Move left
