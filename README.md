Overview simple 2d game where you have to collect food.


## How to run

Download Python.

Install Pygame. To install it, type the following text in terminal: pip3 install pygame

## How to play

You are a snake and you need to collect food.

Controls

⬇️ S Move down

⬆️ W Move up

➡️ D Move right

⬅️ A Move left
